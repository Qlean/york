Используется для создания отступов между элементами

```js
import { Text } from '@qlean/york-web'
;<div>
  <Text preset="h2" htmlTag="h1">
    Аляскинский маламут
  </Text>
  <Separator height={1} />
  <Text preset="caption1" htmlTag="h2" color="grey">
    Материал из Википедии — свободной энциклопедии
  </Text>
  <Separator height={2} />

  <Text>
    Аля́скинский маламу́т (англ. Alaskan Malamute) — достаточно крупная собака
    аборигенного типа, предназначенная для работы в упряжке, одна из древнейших
    пород собак. Своим названием Аляскинский Маламут обязан племени Малемиутов
    (Малемуты, Малемьюты).
  </Text>
</div>
```
